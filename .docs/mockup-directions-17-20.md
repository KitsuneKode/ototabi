## Shared conversion skeleton (all four)

Every direction should ship with:

1. Outcome headline (not feature list)
2. Primary CTA: **Start recording free** + secondary **See demo**
3. One proof row (logos or quote)
4. Three-step workflow
5. Pricing strip (even if “Beta — free”)
6. FAQ + footer watermark

## Recommendation

- **Default pick for Ototabi today:** **19 Patchbay Pro** (evolve current) + add proof/pricing from doc above.
- **If cold traffic from podcast Twitter/YouTube:** **18 Tape Archive** or **19**.
- **If competing on Descript/Riverside “pro” positioning:** **20**.
- **If developer/self-host audience:** **17**.

Do not mix two directions in production — pick one, document in `DESIGN.md`, delete confusion from `/mockups` in marketing.
